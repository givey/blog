Givey is lots of things, but technologically it is an API. All the apps and services we build, like The Burner, FreezeAIDS and even Givey.com are clients of our API. It works with native apps too - our Windows 8 app will be coming out soon.

We’re really keen to get developers using it to power their own charity and donation apps. So far we have a Rails SDK so if anyone is interested in creating an SDK for another language we’ll give you all the help we can.

If you want to get involved check out our API docs and then email Philip.
